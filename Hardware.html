Hardware Components

CPU (Central Processing Unit)

The CPU, or Central Processing Unit, is the primary component of a computer system responsible for executing instructions. It serves as the "brain".

Motherboard

A motherboard is the main circuit board of a computer, providing connectivity and support for many of the crucial components.

ROM (Read-Only Memory)

ROM, a type of memory, stores data permanently and can only be read. It retains data even when the computer is turned off.

RAM (Random Access Memory)

RAM stores data temporarily for quick access by the CPU. It is volatile memory, meaning it loses its contents when the power is turned off.

GPU (Graphics Processing Unit)

A GPU is responsible for rendering images, videos, and animations on a computer screen. It accelerates graphics rendering for better performance.

Cooling Fan

A cooling fan is essential for maintaining optimal temperatures within the computer case, preventing overheating and ensuring proper functionality.

Storage Drive

A storage drive is a device used for storing and retrieving digital information. It can be either a hard disk drive (HDD) or a solid-state drive (SSD).

Power Supply Unit (PSU)

A power supply unit is responsible for supplying power to all components of the computer, converting electrical power from an outlet into usable power for the computer's internal components.

Solid-State Drive (SSD)

An SSD offers faster data access compared to traditional hard disk drives (HDDs). It uses flash memory to store data, resulting in quicker boot times and improved overall system performance.

Hard Disk Drive (HDD)

A hard disk drive (HDD) is a data storage device that uses magnetic storage to store and retrieve digital information. It is the primary data storage device for most computers.
